what are cloud services?

The term cloud services is a broad category that encompas ses the myriad IT resources provided over the internet. The express ion may also be used to describe professional services that support the selection, deployment and ongoing management of various cloud-based resources.


Software as a Service (SaaS) - Salesforce

Software as a servis or SaaS is a way of delivering centrally hosted applications over the Internet as a service.Instead of installing and maintaining software, you simply access it via the Internet, freeing yourself from complex software and hardware management. The provider manages access to the application, including security, av ailability, and performance. SaaS business applications are usually accessed by users using a thin client via a web browser.

Platform as a Service (PaaS) - Cloud 9

Platform as a Service (PaaS) or application platform as a Service (PaaS) or platform base service is a category of cloud computing services that provides a platform allowing customers to develop, run, and manage applications without the complexity of building and maintaining the infrastructure typically associated with developing and launching an app.

Infrastructure as a Service (IaaS) - Amazon

IaaS provides access to computing resource in a virtualised environment, “the Cloud”, across a public connection, usually the internet. In the case of 4 IaaS the computing resource provided is specifically that of virtualised hardware, in other words, computing infrastructure. The definition includes virtual server space, network connections, bandwidth, IP addresses and load balancers. Physically, the pool of hardware resource is pulled from a multitude of servers and networks usually distributed across numerous data centers, all of which the cloud provider is responsible for maintaining. The client, on the other hand, is given access to the virtualised components in order to build their own IT platforms.
